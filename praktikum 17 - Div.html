Pengertian HTML

HTML (Hyper Text Markup Language)

Bahasa Markup atau marka/ penanda yang digunakan untuk membuat menyusun halaman web. HTML berfungsi sebagai struktur dasar atau
kerangka dari sebuah halam web. HTML menggunakan tag dan elemen untuk mengatur tampilan dan perilaku elemen-elemen didalam halaman web

Apa itu Tag?

Tag adalah sebuah penanda awalan dan akhiran dari sebuah elemen di HTML. Fungsi tag digunakan untuk membuat judul,
membuat link, membuat paragraf , heading dan lain-lain. Tag dibuat dengan kurung siku (<...>), lalu di dalamnya berisi nama tag
dan kadang juga ditambahkan dengan atribut.

Apa itu Elemen?

Elemen adalah sebuah komponen yang menyusun dokumen HTML. Elemen kadang juga disebut sebagai node, karena ia merupakan
salah satu jenis node yang menyusun dokumen HTML dalam diagram HTMML tree. Elemen dibentuk dari tag pembuka, isi tag, dan tag penutup.
Kadang juga ditambahkan beberapa atribut.
